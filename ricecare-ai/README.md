# RiceCare AI

RiceCare AI is a mobile-first Human-Centered AI prototype for rice farmers. It provides advisory support for possible rice plant pests and diseases using mock AI logic.

## Tech Stack

React + Vite, JavaScript, React Router, and local mock data.

## Install

```bash
npm install
```

## Run Locally

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Deploy

Deploy the project to Netlify with:

- Build command: `npm run build`
- Publish directory: `dist`

If deploying from the repository root, set the Netlify base directory to `ricecare-ai`.

## Notes

This is a static prototype. It uses mock AI logic and does not perform real disease detection, connect to a backend, or contact real technicians.
